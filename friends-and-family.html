💡How the Apps Work

Each app has its own access codes — TruthPrism uses Prism Packs, News-Distiller uses Distiller Packs. They don't cross over.

Each app also has separate Packs for the Chrome and iOS versions. Prism Packs and Distiller Packs for Chrome are sold via Stripe through the Chrome extension. Packs for iOS are not currently available for sale — F&F can request free iOS Packs as described above.

Both apps include a permanent demo mode that lets anyone see how the analysis works without using a paid check. As a founding-user offer, the first 250 users on each platform receive a small number of free uses (2 for TruthPrism, 5 for News-Distiller) on first install.
