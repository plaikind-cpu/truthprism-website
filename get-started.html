Get Started with TruthPrism

Three ways to use TruthPrism — Chrome extension (recommended), web app, and iOS Share Sheet. The Chrome extension is the easiest: one click on any article and fact-checking begins.

🔑

Step 1 — Get a Prism Code

Required

TruthPrism requires a Prism Code to use. Purchase a Prism Pack at the link below — your code will be emailed to you instantly after payment.

One Prism Code works across all platforms — the web app, iOS Share Sheet, and Chrome extension all share the same balance.

Get a Prism Pack →
💡 No code yet? You get 2 free fact-checks automatically — no account or purchase needed. Just open the app or extension and start checking. When you run out, purchase a Prism Pack to continue.
💡 Purchased codes never expire — use them at your own pace.
🧩

Chrome Extension

One Click
  • 1

    Install from the Chrome Web Store

    Add TruthPrism to Chrome → — click "Add to Chrome" and confirm.

  • 2

    Pin the extension

    Click the puzzle icon in the Chrome toolbar → find TruthPrism → click the pin icon. The TruthPrism icon will appear in your toolbar.

  • 3

    Enter your Prism Code

    Click the TruthPrism icon to open the side panel. Expand Authentication, enter your Prism Code, and click Save.

  • 4

    Click "Check This Page"

    Navigate to any article and click Check This Page in the side panel. TruthPrism will extract the article text and start the analysis automatically.

💡 The extension and web app share the same Prism Code balance — checking an article in either one uses the same check.

🔒 About the Chrome permission warning

When installing, Chrome will warn that TruthPrism can "read and change all your data on all websites." This is standard Chrome language for any extension that reads page content. In practice, TruthPrism only reads the article text on the current page, and only when you explicitly click "Check This Page." It never runs in the background, tracks your browsing, or sends any data other than the article text to our servers for analysis. See our Privacy Policy for full details.

🌐

Web App

Any Device
  • 1

    Open the app

    Go to app.truthprism.app in any browser. Works on desktop, tablet, and iPhone/iPad.

  • 2

    Enter your Prism Code

    Click Authentication to expand it. Paste your Prism Code and click Save. Click Test to confirm it's working. Your code is saved on this device going forward.

  • 3

    Paste an article and check

    Paste any article text into the Check Custom Text field and click Check This Text. Results stream in three steps — summarizing, verifying claims, evaluating context.

⚠️ Paywalled articles (NYT, WSJ, Washington Post) may not load via URL. Copy and paste the article text directly instead.
💡 Add TruthPrism to your iPhone home screen: open in Safari → tap Share → Add to Home Screen.
📱

iPhone & iPad — iOS Shortcut

Share Sheet

Install the TruthPrism shortcut to fact-check any article you're reading with one tap from Safari's Share button.

Option A — One-tap install

Tap the button below on your iPhone or iPad in Safari to install automatically:

📲 Add TruthPrism Shortcut
💡 Tap in Safari on iPhone/iPad — won't work from a desktop browser. Tap "Add Shortcut" when prompted.
Option B — Build it yourself (4 steps)
  • 1

    Open Shortcuts and create a new shortcut

    Open the Shortcuts app → tap + in the top right → tap the name and rename it TruthPrism.

  • 2

    Add "Receive Input from Share Sheet"

    Tap Add Action → search for "Receive" → select Receive Input from Share Sheet → set input type to URLs.

  • 3

    Add "Get URLs from Shortcut Input"

    Tap + → search for "Get URLs" → select Get URLs from Shortcut Input.

  • 4

    Add Text + Open URLs actions

    Tap + → add a Text action → type https://app.truthprism.app?url= → tap the variable icon → select URLs.
    Then tap + → add Open URLs.
    Finally tap the shortcut name → enable Show in Share Sheet → tap Done.

How to use after installing
  1. Open any article in Safari (or any app with a Share button)
  2. Tap the Share button
  3. Scroll down and tap TruthPrism
  4. The app opens and automatically starts the fact-check — no further taps needed
⚠️ Paywalled articles may not load via URL. If you see an error, copy the article text and paste it into Check Custom Text instead.
