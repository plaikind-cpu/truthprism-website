Frequently Asked Questions

Do distillations expire?

No — your distillations never expire. Use them at whatever pace suits you, whether that's one article a day or ten in a week.

Does one code work across all platforms?

Yes — your Distiller code and its balance works across the web app, iOS Share Sheet shortcut, and Chrome extension. They all draw from the same pool.

How quickly do I get my code after purchase?

Your Distiller code is emailed to you within seconds of completing payment. Check your spam folder if it doesn't arrive within a minute or two.

What counts as one distillation?

Each article analysis uses one distillation — regardless of article length or report type. Distillations are only consumed on successful completion.

Can I buy more when I run out?

Yes — just purchase another Distiller Pack. You can use the same email address and the new code will start fresh.

Can I use a TruthPrism Prism Pack code?

Yes — existing TruthPrism Prism Pack codes also work in News-Distiller, drawing from the same check balance.

What payment methods are accepted?

All major credit and debit cards via Stripe. Payment is handled entirely by Stripe — we never see your card details.

Is there a refund policy?

If you have an issue with your purchase, contact support@truthprism.app and we'll make it right.
