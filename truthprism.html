How It Works
Three steps, 20–40 seconds. Powered by Claude AI and Brave Search.
1

Summarize Claude AI

Claude reads the article and extracts a concise executive summary plus 3–6 of the most consequential factual claims — calibrated to article complexity. Simple articles get 3 claims; complex analytical pieces get up to 6.

2

Verify Claims Brave Search

All claims are verified simultaneously using real-time Brave Search. Claude generates a targeted search query for each, retrieves current web sources, and evaluates the evidence — assigning a verdict and sourced finding for each claim.

3

Evaluate Context Claude AI

Beyond the facts, Claude evaluates how the article presents information — flagging charged language used as established fact, unsourced motivational claims, missing counterarguments, and one-sided framing. This produces the Context Score and the Context & Framing section.

Claim Verdicts
Each verified claim gets one of five verdicts based on real-time evidence.
SupportedMultiple credible sources confirm the claim.
Partially SupportedRoughly accurate but with important caveats or minor inaccuracies.
UnverifiedSearch results don't address the claim directly — treated as neutral, not false.
ContestedSources actively disagree with the claim.
FalseCredible sources directly contradict the claim.

🧩 Chrome Extension — Recommended for Desktop
The most convenient and powerful way to use TruthPrism — reads the page directly from your browser, including paywalled articles you subscribe to.

Now live on the Chrome Web Store

One click on any article and fact-checking begins — no copy-paste, no switching tabs. Works on NYT, WSJ, and any paywalled site you subscribe to, because the extension reads what's already on your screen.

🧩 Install TruthPrism for Chrome
💡 Works on Chrome and Edge on any desktop or laptop. Click the TruthPrism icon in your browser toolbar while on any news article to begin.

Your Prism Pack access code works in the extension — enter it once in the Authentication section and it's saved permanently.

🌐 Web App — Backup Option
If the Chrome extension isn't ideal for your situation, the web app provides the same analysis on any browser.

Use TruthPrism on the web

Open app.truthprism.app in any browser, paste an article URL or text, and get the same structured fact-check as the extension. Your Prism Pack code works in both.

💡 The Chrome extension is recommended for daily use — it reads the page directly, including paywalled sites you subscribe to. The web app is a useful fallback when Chrome isn't available.
