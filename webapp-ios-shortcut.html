iOS Safari Shortcut for the Web App

Send articles to the TruthPrism or News-Distiller web app with one tap from Safari's Share Sheet on iPhone or iPad.

What this does: sets up an iOS Shortcut that opens the web app with the article URL pre-loaded. This is for the web apps (app.truthprism.app and app.news-distiller.com), not the iOS Safari apps. Setup takes 3 minutes per app.

🔍 TruthPrism — Web App Shortcut

Fact-check any article from Safari with one tap.
Option A — One-tap install

Tap this on your iPhone or iPad in Safari:

📲 Add TruthPrism Shortcut
💡 Must be tapped in Safari on iPhone or iPad — won't work from a desktop browser. Tap "Add Shortcut" when prompted.
Option B — Build it yourself (4 steps)
  • 1

    Open Shortcuts and create a new shortcut

    Open the Shortcuts app → tap + in the top right → tap the name and rename it TruthPrism.

  • 2

    Add "Receive Input from Share Sheet"

    Tap Add Action → search for "Receive" → select Receive Input from Share Sheet → set input type to URLs.

  • 3

    Add "Get URLs from Shortcut Input"

    Tap + → search for "Get URLs" → select Get URLs from Shortcut Input.

  • 4

    Add Text + Open URLs actions

    Tap + → add a Text action → type https://app.truthprism.app?url= → tap the variable icon → select URLs.
    Then tap + → add Open URLs.
    Finally tap the shortcut name → enable Show in Share Sheet → tap Done.

How to use: open any article in Safari → tap Share → tap TruthPrism → the web app opens and analysis starts automatically.
⚠️ Paywalled articles (NYT, WSJ, etc.) may not load via URL. If you see an error, copy the article text and paste it into the web app's Check Custom Text field instead.

📲 Tip — Add the Web App to Your Home Screen

For fastest access, you can also add either web app to your home screen so it opens like a native app:

Open Safari → go to app.truthprism.app or app.news-distiller.com → tap the Share button → tap "Add to Home Screen" → tap Add.

💡 Must be done from Safari — Chrome on iPhone doesn't support adding to home screen.
