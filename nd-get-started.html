Get Started with News-Distiller

Use News-Distiller on any device — browser, Chrome extension, or iPhone Share menu. Setup takes about 2 minutes.

🌐

Web App

Any Browser
  • 1

    Open News-Distiller

    Go to app.news-distiller.com in any browser on any device. No installation required.

  • 2

    Enter your access code

    News-Distiller uses the same access codes as TruthPrism. Enter your existing Prism Pack code and tap Save Code — it's remembered on your device. No code yet? Get a Prism Pack →

    💡 TruthPrism Prism Pack codes work in News-Distiller now — one purchase covers both apps.
    🔜 Distiller Packs coming soon — lower-cost access codes just for News-Distiller.
  • 3

    Paste an article URL or text

    Copy the URL of any news article and paste it into the URL field, then tap Distill. Or paste the article text directly into the text field. The summary streams in real time as it's generated.

    ⚠️ Paywalled articles (NYT, WSJ, Washington Post, etc.) cannot be retrieved by URL. Copy and paste the article text directly instead — this always works.
  • 4

    Read your summary

    News-Distiller returns an Executive Summary for a quick read, and a Full Report with the complete in-depth breakdown. Both can be saved as text files. The political lean indicator is shown with a high bar — Center is the default unless the text provides clear evidence of lean.

📲

Add to Your Home Screen

For fastest access, add News-Distiller to your home screen so it opens like a native app.

  • iOS

    iPhone & iPad (Safari)

    Open Safari → go to app.news-distiller.com → tap the Share button (box with arrow) → tap "Add to Home Screen" → tap Add. Must use Safari — Chrome on iPhone does not support this.

  • Mac

    Desktop (Chrome)

    Open Chrome → go to app.news-distiller.com → click the install icon in the address bar (screen with down arrow) → click Install. News-Distiller will open as its own window.

iPhone & iPad — Share Sheet

iOS Shortcut

Add News-Distiller to your iOS Share menu so you can distill any article with one tap from Safari or any app. Setup takes about 3 minutes.

Option A — One-tap install

Tap the button below on your iPhone or iPad to install the shortcut automatically:

📲 Add News-Distiller Shortcut
💡 If the iCloud link doesn't open automatically, tap it in Safari on your iPhone — it won't work from a desktop browser.

Option B — Build it yourself (4 steps)

  • 1

    Open Shortcuts and create a new shortcut

    Open the Shortcuts app on your iPhone → tap + in the top right → tap the shortcut name and rename it "News-Distiller".

  • 2

    Add a "Receive" action

    Tap "Add Action" → search for "Receive" → select "Receive Input from Share Sheet" → set the input type to URLs. This enables the shortcut to appear in your Share menu.

  • 3

    Add a "Get URLs" action

    Tap + → search for "Get URLs" → select it → confirm the input is set to Shortcut Input.

  • 4

    Add an "Open URLs" action with the News-Distiller base URL

    Tap + → search for "Text" → select the Text action → type: https://app.news-distiller.com?url= then immediately tap the variable icon and select URLs (from step 3). Then add one more action: "Open URLs" and set it to open the Text variable. Tap Done.

    ⚠️ Some paywalled articles (NYT, WSJ, etc.) cannot be retrieved via URL even from the Share sheet. For those, copy and paste the article text directly into the web app.

Using It

  1. Open any article in Safari or another app
  2. Tap the Share button (box with arrow pointing up)
  3. Scroll and tap News-Distiller
  4. The app opens with the article URL — distillation starts automatically
